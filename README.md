# 1210 MethodOS™

**Enterprise Domain Specification for Hospitality Engineering, Consulting, Knowledge Architecture and AI-native Operations.**

1210 MethodOS™ ist ein versioniertes Repository für Hospitality Engineering. Es verbindet Department 66, StiegerConsult, CaaS, D66 Academy, Cocktail-Analyst und Forensischer Barkeeper™ zu einem gemeinsamen Referenzsystem.

## Quick Start

```bash
git clone <repo-url>
cd 1210-methodos
pip install -r requirements.txt
mkdocs serve
```

## Repository-Bereiche

| Bereich | Zweck |
|---|---|
| `docs/` | Menschlich lesbare Spezifikationen |
| `schemas/` | JSON, SQL, OpenAPI, GraphQL |
| `ontology/` | RDF/OWL/Cypher/SPARQL |
| `ai/` | RAG, Agenten, MCP, Evaluationen |
| `tests/` | Validierung und Qualitätssicherung |
| `sdk/` | spätere SDKs |
| `tools/` | Hilfswerkzeuge |
