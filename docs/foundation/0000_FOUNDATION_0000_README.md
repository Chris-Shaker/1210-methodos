# 1210 MethodOS™

> **The Operating System for Hospitality Engineering**

## Willkommen

Dieses Repository ist die zentrale Wissens- und Methodenbasis von **1210
MethodOS™**.

Es verfolgt das Ziel, ein wissenschaftlich aufgebautes, AI-natives
Referenzsystem für Hospitality Engineering zu schaffen. Das Repository
dient als *Single Source of Truth* für Architektur, Standards,
Frameworks, SOPs, Forschung, Implementierungen und Produkte.

## Vision

Eine weltweit anerkannte Referenzarchitektur für Hospitality Engineering
entwickeln, vergleichbar mit ITIL, dem Toyota Production System oder
TOGAF -- jedoch speziell für die Hospitality-Branche.

## Mission

MethodOS verbindet:

-   Hospitality
-   Betriebswirtschaft
-   Psychologie
-   Informationsdesign
-   Wissensmanagement
-   Softwarearchitektur
-   Künstliche Intelligenz
-   Continuous Improvement

zu einem konsistenten System.

## Leitprinzip

Nicht Unternehmen sind das eigentliche Produkt.

**Die Methode ist das Produkt.**

Department 66, StiegerConsult, Cocktail-Analyst, CaaS und die D66
Academy sind Referenzimplementierungen derselben Methodik.
